# Strategy

This folder holds durable strategic knowledge.

Use it for strategy that should remain valid beyond a single project, sprint, or quarter.

## What belongs here

- Ideal customer profile definitions
- Positioning and category choices
- Strategic theses and long-term bets
- Enduring go-to-market frameworks
- Durable market and business model choices

## What does not belong here

- Current-quarter priorities
- Temporary campaign focus
- In-flight initiative plans
- Time-bound operating assumptions

Those belong in `Knowledge/Operating System/Active Context/`.

## Rule of thumb

If a note should still be useful in 6-18 months, it probably belongs here.
If it is mainly about what is true right now, it probably belongs in Active Context.

## Suggested note types

- ICP notes
- Positioning notes
- Strategic bet notes
- Market selection notes

## Linking

- Link strategy notes to supporting concepts in `Knowledge/Foundations/Concepts/`.
- Link current applications of strategy from `Knowledge/Operating System/Active Context/` back to the canonical strategy note here.
