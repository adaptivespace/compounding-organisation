# Glossary

Living reference of terms and definitions in alphabetical order. The LLM checks this before using any technical term. Updated on every ingest that introduces new or refined terminology.

---
## How to Read This Glossary

Each entry follows this format:

**Term** *(canonical form)*
: Definition. Usage notes. Related terms.
- Preferred: `term` / Avoid: `deprecated term`
- See also: *(link to a relevant page)*

---

**Compounding organisation**
: An organisation that systematically builds durable, shared understanding from its accumulated evidence, decisions, and operating experience — rather than letting context leak or requiring repeated reconstruction. The product category this knowledge base is designed to support.
