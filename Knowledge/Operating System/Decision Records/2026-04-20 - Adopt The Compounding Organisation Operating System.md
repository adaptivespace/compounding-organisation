**context:** We are establishing the organisation's operating model and need a consistent way to capture strategy, decisions, active context, metrics, and playbooks.

**facing:** Fragmented knowledge and inconsistent ways of working, leading to loss of context and poor decision traceability.

**we decided for:** We will start using the Compounding Organisation operating system as the shared framework for organising knowledge and running the organisation.

**and neglected:** We are choosing against having no common operating model and against high-touch centralised knowledge management.

**to achieve:** A compounding knowledge base that enables clearer decisions and helps the organisation move forward using a shared, actionable context.

**accepting that:** A text-based knowledge base may run into scalability issues over time.
