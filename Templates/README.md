# Templates

Reusable templates for the main note and deliverable types in this vault.

## Structure

```text
Templates/
  Knowledge/
    Concept.md
    Decision Record.md
    Playbook.md
  Outputs/
    Memo.md
    Report.md
  Research/
    Research Capture.md
```

## Usage

- Use `Templates/Knowledge/` when creating curated notes in `Knowledge/`..

Keep templates short, durable, and easy to fill in.
